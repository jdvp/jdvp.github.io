Statistexts Release Notes

Release notes for the Statistexts app found on Google Play : Statistexts

1.63
October 5, 2018
Re-added the internet permission that was removed in release 1.60 (May 2018 release) in order to fix graphing functionality that relies on the Google Charts APIs.

1.62
October 4, 2018
Updated app icon to be an adaptive icon. Updated library versions to increase stability. Updated android support libs to androidx.

1.61
May 29, 2018
Fixed crash caused by trying to analyze texts in the background incorrectly --- Removed ads! Also removed analytics, which I wasn't really actively monitoring anyway. Now, the app no longer uses internet permissions at all and should be GDPR compliant. Updated libraries used and optimized app size to decrease app size from 2.18 MB to 1.32 MB

1.6
May 28, 2019
Removed ads! Also removed analytics, which I wasn't really actively monitoring anyway. Now, the app no longer uses internet permissions at all and should be GDPR compliant. Updated libraries used and optimized app size to decrease app size from 2.18 MB to 1.32 MB

1.45
November 8, 2016
NEW OPTIONS
1. Can now choose how many hours between text messages counts as a reply (any more time than the selected counts as a new conversation)
2. Can now hide and unhide contacts. Hiding is useful so you don't always have to see statistics about every single contact (i.e. your bank's texts etc.). Do this by long pressing a contact in the contact list. Unhiding is done in the settings menu

1.40
November 5, 2016 Sped up analysis of messages
This includes analysis done in the app when loading for the first time and also analysis of messages as they come in so that a user doesn't have to wait for new analysis results
